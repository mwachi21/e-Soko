# Gettting Started Guide

## Django Installation

```bash
 python -m venv .venv;
 .\.venv\Scripts\activate;
 pip install django;
 ```

 ```bash
 django-admin startproject blkmkt .
 ```

## Start Django Server

```bash
python manage.py runserver
```

## Migration
```bash
python manage.py migrate
```

## Create Super User
```bash
python manage.py createsuperuser
```
## Accounts
```bash
python manage.py startapp accounts
```
