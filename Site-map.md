# e-Soko

Project Name : e-Soko,
Front-End : Svelte,
Backend : Flask,
Database : SQALchemy,
Day Started : 27th/Jan/2026

Mission : Any User To Sell or Buy

## User/Account Features

* Account Creation (OAuth - Google Sign In)
* User Login
* User Dashboard
* Users To post products
* Account Deletion

## Products Features

* Name
* Price
* Description
* Condition (New, Refurbished, Used)
